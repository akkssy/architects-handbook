üöÄ Quick Start

Get Started in Minutes

Choose your platform and preferred LLM provider. Both local and cloud options available.

1

Install Cognify CLI

Install directly from PyPI (recommended).

pip install cognify-code
2

Choose Your LLM (Local OR Cloud)

Option A: Local with Ollama (free, private, offline)

# Install Ollama from ollama.ai, then: ollama pull deepseek-coder:6.7b

Option B: Cloud with Groq (fast, free tier)

# Set your API key (get free at console.groq.com) export GROQ_API_KEY=your_key_here
3

Start Using!

Review code, generate functions, or start chatting.

# With local Ollama cognify review src/app.py # Or with Groq (cloud) cognify review src/app.py --provider groq
1

Install Prerequisites

The extension requires the CLI backend.

pip install cognify-code
2

Install VSCode Extension

Install from the VSCode marketplace or build from source:

# From VSCode Marketplace (coming soon) # Or build from source: cd vscode-extension && npm install npm run compile && npx vsce package
3

Configure Provider

Open VSCode Settings (Cmd+,) and search for "Cognify":

  • cognify.provider: ollama, groq, openai, google, cerebras
  • cognify.model: deepseek-coder:6.7b, llama3, gpt-4o, etc.
4

Start Using!

Use keyboard shortcuts or the command palette:

  • Cmd+Shift+R ‚Äî Review current file
  • Cmd+Shift+G ‚Äî Generate code
  • Cmd+Shift+C ‚Äî Open chat panel
üè†

Local: Ollama (Recommended for Privacy)

100% free, runs on your machine, no internet required.

# Install from ollama.ai, then pull a model ollama pull deepseek-coder:6.7b # Use with CLI cognify review code.py --provider ollama --model deepseek-coder:6.7b
‚ö°

Cloud: Groq (Recommended for Speed)

Ultra-fast inference, generous free tier.

export GROQ_API_KEY=your_key_here cognify review code.py --provider groq --model llama3-70b-8192
üîÆ

More Cloud Options

OpenAI, Google Gemini, Cerebras, OpenRouter are all supported.

# OpenAI export OPENAI_API_KEY=your_key cognify chat --provider openai --model gpt-4o # Google Gemini export GOOGLE_API_KEY=your_key cognify chat --provider google --model gemini-pro

Command Reference

üìö Knowledge Base

Build a persistent knowledge base that grows smarter over time. Automatically included in AI responses.

üí°

How It Works

Knowledge is stored as markdown files in .cognify/knowledge/ organized by category. When you chat with the AI, relevant knowledge is automatically retrieved and included in the context.

üìù

Save Knowledge

Save patterns, architectural decisions, or important code snippets.

cognify knowledge save "Repository Pattern" \
--content "Use repository pattern for data access..." \
--tags architecture --tags patterns \
--category architecture
üîç

Search & List

Find relevant knowledge or list all entries.

cognify knowledge search "authentication" --limit 5
cognify knowledge list --category patterns
cognify knowledge stats
